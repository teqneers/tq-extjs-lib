TQ Ext JS Component Library

  • API Documentation
  • Jasmine Tests (Test can be run with PhantomJS using phantomjs ./test/run-jasmine.js http://<your host>/<path to tq-extjs-lib>/test/)

Uses the following third-party libraries for unit testing

  • Jasmine 1.3.1
  • Jasmine-jQuery 1.5.0
  • jQuery 1.9.1
  • run-jasmine.js test-runner for PhantomJS
